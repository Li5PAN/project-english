## 技术细节

### wordService.js 提供的函数

```javascript
// 获取学生单词列表（10个）
getStudentWords(options)

// 获取学生单词（带详细信息和进度回调）
getStudentWordsWithDetails(options, onProgress)

// 获取任务单词
getTaskWords(taskId)
getTaskWordsWithDetails(taskId, onProgress)

// 搜索单词
searchWord(keyword)

// 收藏功能
addToFavorites(word)
removeFromFavorites(word)

// 记录进度
recordProgress(progress)
```

### 数据结构

每个单词包含：
- `word` - 单词
- `ukPhonetic` - 英式音标
- `usPhonetic` - 美式音标
- `ukSpeech` - 英式发音 URL
- `usSpeech` - 美式发音 URL
- `meanings` - 释义数组
- `examples` - 例句数组
- `wordForms` - 词形变化数组
- `webMeanings` - 网络释义数组
